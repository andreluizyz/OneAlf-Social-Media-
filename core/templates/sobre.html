Sobre o Projeto

O OneAlf é um projeto de rede social criado com o propósito de estudo, prática e aperfeiçoamento das tecnologias do Django, Python e Tailwind CSS. O sistema permite que usuários se registrem, criem perfis, editem suas informações e compartilhem publicações com imagem e texto.

Desenvolvido por Alffex, o projeto visa simular o funcionamento de uma rede social moderna, permitindo entender na prática conceitos fundamentais como:

  • Autenticação de usuários
  • Relacionamentos entre modelos
  • Upload de imagens e arquivos
  • Views protegidas com login
  • Renderização dinâmica com templates
  • Boas práticas com formulários e validações

Todo o projeto está disponível publicamente no GitHub, para consulta, aprendizado ou contribuição da comunidade.
